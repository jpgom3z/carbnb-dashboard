# Carbnb Dashboard

A mobile-first delivery tracking dashboard for managing and monitoring deliveries in real-time.

## Features

- Real-time delivery tracking
- Filter deliveries by status (All, Active, Landed, Delayed)
- Car assignment for each delivery
- WhatsApp integration for driver contact
- Mobile-optimized interface
- Responsive design (mobile, tablet, desktop)

## Usage

Open `index.html` in a web browser to view the dashboard.

## Mobile Optimization

The interface is designed primarily for mobile devices with touch-friendly buttons and appropriately sized text for easy readability on smartphones.
